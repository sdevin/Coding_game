But du scénario :
	amener le personnage au point d'arrivé (comptoir d'enregistrement)
	
Données initiales :
	Nom perso par défaut : toto
	Position perso x par défaut : 1
	Position perso y par défaut : 5
	Position arrivée x : 10
	Position arrivée y : 7
	
Nom du fichier de communication pour les commande : 
	/tmp/cmd
	
Commandes possibles :
	persoName direction
avec direction = r (droite), l (gauche), u (haut) ou d (bas) 
exemples de commande :
toto d
toto u
	end
a mettre a la fin du code

Fichier de configuration :
	persoName (String) : pour changer le nom du personnage
	randPos (true ou false) : si a true la position initiale du personnage sera aléatoire
	posPersoX (int) : position x (numéro de case) initiale du personnage (non lu si randPos=true)
	posPersoY (int) : position y (numéro de case) initiale du personnage (non lu si randPos=true)
	
Si randPos = true (niveau bonus)
	Le personnage aura une position de départ aléatoire (dans les limites du quadrillage)
	Cette position sera envoyée dans le fichier /tmp/posPerso sous le format :
nomPerso posX posY
ex : toto 3 4
